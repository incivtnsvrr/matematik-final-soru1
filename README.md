# matematik-final-soru1
matematik final soru1
istanbul kent üniversitesi maatematik ödevi için c++ programa dili kullanarak geliştirilmiş ,ikinci dereceden fonksiyonların köklerini hesaplayan bir uygulamadır.
öğrenci bilgileri:
ad soyad: inci vatansever
öğrenci no: 2507020074
bölüm : bilgisayar mühendisliği 
programın çalışma mantığı : İlk olarak kullanıcıdan ax^2+bx+c ikinci dereceden denkleminin a, b ve c katsayılarını girmesi istendi. Girilen bu değerlere göre programda delta (delta = b^2− 4ac) hesaplandı.

Hesaplanan delta değeri için üç farklı durum ele aldım: delta küçüktür 0, delta eşittir 0 veya  delta büyüktür sıfır  olabilir.

Delta değeri 0’dan küçükse, denklemin gerçek kökü olmadığı belirleriz. Delta 0’a eşit ya da 0’dan büyük olduğunda ise, uygun formüller kullanılarak denklemin kökleri hesaplarız ve sonuç kullanıcıya görterilmiş olur.
